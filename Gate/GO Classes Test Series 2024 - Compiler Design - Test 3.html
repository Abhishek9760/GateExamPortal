Q8 -

A node p in a control flow graph(CFG) N dominates a node q if every path from the entry node to q goes through p. We say that node p is a dominator of node q.
The dominator set of node q, DOM(q), is formed by all nodes that dominate q. By definition, each node dominates itself, therefore, q∈DOM(q)

  • p dominates q, written p≤q, i.e. p∈DOM(q).
  • p properly (strictly) dominates q, written p lt;q if p≤q and p≠q.
  • p immediately (or directly) dominates q, if p lt;q and there is no node t such that p lt;t lt;q.

We say p is the immediate dominator of q and write p=IDOM(q)
